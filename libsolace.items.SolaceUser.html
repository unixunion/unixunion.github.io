libsolace.items.SolaceUser module¶

class libsolace.items.SolaceUser.SolaceUser[source]¶

Manage a Solace Client User

This plugin manages Client Users within Solace. Typically you should invoke this plugin via L{SolaceAPI.SolaceAPI}.

Please see L{plugin.Plugin} for how plugins are instantiated and used.

Shutdown the user, this method will be called by anything decorated with the @shutdown decorator. The kwarg shutdown_on_apply needs to be either True or ‘u’ or ‘b’ for this method to fire.
